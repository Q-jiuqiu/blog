# 过滤器--filters (opens new window)

# 全局过滤器

vue.filter("过滤器名称",function(value){
// ……
})

# 局部过滤器

filters: {
  "过滤器名称": function (value) {
    // ……
  }
}

过滤器的使用：

{{ a | 过滤器名称 }}
或者
<div v-bind:id="a | 过滤器名称 "></div>

当全局过滤器和局部过滤器重名时，会采用局部过滤器。
