# 事件处理

<template>
  <div class="eventsHandle">
    <button @click="funA('这是一个：', $event)">访问DOM节点</button>
  </div>
</template>

<script>
export default {
  data() {
    return {};
  },
  methods: {
    funA(message, event) {
      console.log(message, event.target);
    }
  }
};
</script>

可以用特殊变量 $event 访问原始的 DOM 事件
