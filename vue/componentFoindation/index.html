# 组件基础

  1. 全局组件

    // ---在main.js中---
// 注册全局自定义组件
Vue.component("allButton", {
  data() {
    return { count: 0 };
  },
  template: '<button @click="count++">{{count}}</button>'
});

new Vue({
  el: "#app"
});

  2. 局部组件

    --- 第一种写法 ---
<script>
export default {
  components: {
    myButton: {
      template: '<button @click="add">{{count}}</button>',
      data() {
        return {
          count: 1
        };
      },
      methods: {
        add() {
          this.count++;
        }
      }
    }
  }
};
</script>

--- 第二种写法 ---
<script>
import myA from "./components/A";
export default {
  data() {
    return { param1: "jhh", param2: "123" };
  },
  components: {
    myA
  }
};
</script>

因为你每用一次组件，就会有一个它的新实例被创建，所以在复用组件的时候他们自己的值(data)是相互独立的。

# 为什么data是一个函数?

组件是一个具有单独自身功能模块的封装，这个模块有属于自己的HTML模板，也应该有属于自己的数据data，组件也有自己的方法methods。data属性必须是一个函数，并且这个函数返回一个对象，在对象内部保存数据
