要将地图包括在网页中，将需要三件事：(构建Openlayers)

  1. 包括OpenLayers

  2. <div> 地图容器

    <div id="map" class="map"></div>

    注意地图容器一定要定义大小，因为地图在此容器中显示，若没有定义大小此容器的高度为0，地图将不会显示

  3. 用JavaScript创建一个简单的地图

    new Map({
    target: "map", // 对应页面里 id 为 map 的元素,表示地图挂在在哪个节点上
    layers: [
        // 图层，数组用于定义地图中可用的图层列表
        new TileLayer({
            source: new OSM() // 图层数据源，此处使用的时官方的默认地图
        })
    ],
    view: new View({
        // 地图视图
        projection: "EPSG:4326", 
        // 投影，有EPSG:4326和EPSG:3857，相当于通过一系列算法将地球这个不规则的椭圆体投影到平面上
        center: [118.120581, 24.476726], 
        //经纬度坐标（厦门） 地图的视图中心
        minZoom: 10, // 地图缩放最小级别
        zoom: 11 //地图缩放级别
    })
});
